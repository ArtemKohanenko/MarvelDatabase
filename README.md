# General info

This project is being developed as part of a course from Effective.
This is frontend application that provides information about Marvel heroes.

## Technologies

- Node.js: 20.11.1
- React: 18.2.0
- Typescript: 5.2.2

## Setup

```
cd MarvelDatabase
npm install
npm run dev
```
